# WPGraphQL for Advanced Custom Fields

WPGraphQL for Advanced Custom Fields is a FREE open-source WordPress plugin. 
The code is available on [Github](https://github.com/wp-graphql/wp-graphql-acf). Support and feature requests are handled through [Github issues](https://github.com/wp-graphql/wp-graphql-acf/issues). For general questions about the plugin, visit the [WPGraphQL Slack}(https://wp-graphql.slack.com/) ([join here](https://wpgql-slack.herokuapp.com)) or the WPGraphQL Spectrum community.
